# 🗓️ 인생 타임라인 (Life Timeline)

생년월일을 기준으로 **인생 100년(기본 90년)을 한 화면에** 펼쳐 보는 무한 캔버스 캘린더입니다.
한 칸은 인생의 한 주(週)이며, 살아온 주는 채워지고 다가올 주는 비어 있습니다.
인생의 진척도를 한눈에 보면서 일정을 기록하세요.

> 순수 **HTML / CSS / JavaScript** 로만 작성되었습니다. 외부 라이브러리·빌드 도구·의존성이 전혀 없습니다.

## ✨ 기능

- **생년월일 입력** — 첫 실행 시 생년월일과 기대 수명을 입력하면 캘린더가 생성됩니다.
- **100년치 캘린더** — `1년 = 52주`, `90년 = 4,680칸`. 한 칸이 곧 한 주입니다.
- **인생 진척도** — 살아온 날·주·나이와 함께 퍼센트를 상단 바에 표시합니다.
- **무한 캔버스** — 드래그로 이동, 마우스 휠(또는 두 손가락 핀치)로 확대/축소.
- **일정 관리** — 아무 칸이나 클릭하면 그 주의 일정을 보고, 특정 날짜에 일정을 추가/삭제할 수 있습니다. 일정이 있는 주에는 점 표시가 나타나며, "다가오는 일정"에서 클릭하면 해당 주로 이동합니다.
- **로컬 저장** — 모든 데이터는 브라우저 `localStorage` 에 저장됩니다. 서버가 필요 없습니다.

## 🎨 시각 규칙

| 칸 모양 | 의미 |
| --- | --- |
| 진한 칸 | 이미 지나온 주 (과거) |
| 보라색 칸 | 지금 살고 있는 이번 주 |
| 빈 칸 | 아직 오지 않은 주 (미래) |
| 가운데 점 | 일정이 등록된 주 |

## ▶️ 사용법

1. `index.html` 을 브라우저에서 열기만 하면 됩니다. (별도 서버 불필요)
2. 처음 실행하면 생년월일과 기대 수명을 입력합니다.
3. 캔버스를 드래그/확대하며 둘러보고, 칸을 클릭해 일정을 추가하세요.
4. 우측 상단 ⚙ 버튼으로 언제든 생년월일·기대 수명을 바꿀 수 있습니다.

## 🚀 GitHub Pages 배포

이 저장소는 정적 파일만 포함하므로 그대로 배포할 수 있습니다.

1. GitHub 저장소에 푸시합니다.
2. **Settings → Pages → Source** 를 `main` 브랜치 / `root` 로 지정합니다.
3. 잠시 후 `https://<사용자명>.github.io/<저장소명>/` 에서 접속됩니다.

> `.nojekyll` 파일이 포함되어 있어 Jekyll 처리 없이 정적 파일이 그대로 서빙됩니다.

## 📁 파일 구조

```
index.html   — 마크업 / 레이아웃
styles.css   — 미니멀 테마 스타일
app.js       — 캔버스 렌더링 · 카메라 · 일정 로직 (의존성 없음)
.nojekyll    — GitHub Pages용
```
